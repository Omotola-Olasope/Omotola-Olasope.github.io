Data Migration

Large-Scale On-Premise to Cloud Data Migration with Aws Cloudformation, File Gateway and Cloudwatch Alarms

This project involved the successful migration of a large organization's on-premise data to the cloud. Leveraging the power of AWS CloudFormation, I created a secure and scalable infrastructure, including: VPC Base: Defined the network infrastructure for deploying resources. Windows Instance: Established an on-premise machine for file server access. File Gateway: Facilitated seamless data transfer from the on-premise network. NFS File Share: Received migrated data in the cloud. CloudWatch Alarm: Triggered alerts for exceeding data transfer thresholds.

Streaming Pipeline

Dynamic Real-Time Streaming Pipeline Architecture

In this project, I restructured a Python script into a dynamic Lambda function, leveraging AWS for the development of a real-time financial data streaming pipeline. The orchestration is handled through an IAM role endowed with Admin privileges, ensuring seamless execution. Key components include the establishment of a DynamoDB table for data storage, the implementation of automated Lambda triggers using CloudWatch, and the incorporation of proactive monitoring through AWS SNS notifications. This optimized architecture on AWS enhances the efficiency and responsiveness of the financial data streaming process.
